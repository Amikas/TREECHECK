
# TREECHECK

Checks if a binary search tree (BST) from a file is an AVL tree.

## How to Run

1. Open terminal in this folder.
2. Run: python main.py
3. When prompted enter: numbers.txt


## Input File

- One integer per line.

## Output

- Preorder traversal
- Tree height
- `True` if AVL, `False` if not
